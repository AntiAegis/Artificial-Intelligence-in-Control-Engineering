\section{Problem}
Given a single-variable function
$$
y(x) = 4x^4 - 5x^3 + exp^{-2x} - sin(x) - 3cos(x)
$$
subject to $x \in [0, 5]$, use the Genetic Algorithm to find the global maximum of the function $y(x)$.
